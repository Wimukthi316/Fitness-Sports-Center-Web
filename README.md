## 📖 Project Overview

Fitness Sports Center Web App is a modern, high-impact promotional website built to represent a premium fitness brand and attract new members through strong visual storytelling, responsive design, and conversion-focused user experience.

Developed as part of a **Web Development Internship Evaluation**, this project demonstrates production-ready frontend engineering practices, polished UI/UX implementation, and enterprise-style cloud deployment.

## ⚙️ Core Features

- 🦾 **Hero Section:** Bold fitness branding, compelling call-to-actions, and animated headline experience.
- 🏢 **About Section:** Brand story, trust indicators, and premium facility showcase with engaging layout.
- 🧩 **Services Section:** Structured service cards with responsive grid presentation and user-friendly discovery.
- 📬 **Contact Form:** Interactive inquiry form with smooth feedback flow and polished visual states.

## 🏆 Bonus Features Achieved

- ✅ **Dark/Light Mode Toggle:** Custom CSS overrides for seamless, accessible theme switching.
- ⭐ **Search/Filter Functionality:** Real-time dynamic filtering in the Services section.
- ✅ **Premium Animations:** Scroll-triggered staggered reveals and text effects powered by Framer Motion.
- ⭐ **Advanced Form Handling:** Regex-based validation with dynamic error and success feedback.
- ✅ **Enterprise Deployment:** Automated CI/CD pipeline using Azure Static Web Apps with GitHub Actions.

## 💻 Local Setup Instructions

### 1) 📥 Clone the Repository

```bash
git clone https://github.com/Wimukthi316/Fitness-Sports-Center-Web.git
```

### 2) 📂 Navigate to Frontend

```bash
cd Fitness-Sports-Center-Web/frontend
```

### 3) 📦 Install Dependencies

```bash
npm install
```

### 4) ▶️ Run Development Server

```bash
npm run dev
```
